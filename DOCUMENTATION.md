# NexaWork - Developer Documentation

## Technical Stack
- **Backend**: Laravel 10 (PHP 8.1+)
- **Mobile**: React Native (Expo SDK 50+)
- **Database**: MySQL / SQLite (Development)
- **Authentication**: JWT (JSON Web Tokens)
- **Real-time**: Polling (Optimized)
- **Notifications**: Expo Push Notifications API

## Key Features
1. **Job Board**: Premium UI for listing, searching, and applying for jobs.
2. **Linkup (Professional Networking)**: User discovery based on industry/skills.
3. **Real-time Messaging**: Multi-media chat (Image/File support).
4. **Admin Suite**: Comprehensive dashboard for user moderation, job management, and system snapshots.
5. **Security**: Encrypted passwords, Admin-only middleware, and strict profile concealment for admins.
